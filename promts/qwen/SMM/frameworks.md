| **HOOK, STORY, OFFER** | Hook → Story → Offer                                    | Зацепка → История → Предложение                       | Структура продающего поста: цепляем, вовлекаем эмоцией, даём оффер                                  |

## 📊 Контент и презентации

### Базовые структуры

| Фреймворк               | Расшифровка                                                   | Комментарий                                                                   |
| ----------------------- | ------------------------------------------------------------- | ----------------------------------------------------------------------------- |
| **PPP**                 | Preview → Promise → Proof (Обзор → Обещание → Доказательство) | Коротко о содержимом → что получит читатель → подтверждение слов              |
| **FOMO**                | Fear Of Missing Out (Страх упустить выгоду)                   | Триггер срочности: «только сейчас», «ограничено», «последний шанс»            |
| **5W+H**                | Who/What/When/Where/Why/How (Кто/Что/Когда/Где/Почему/Как)    | Чек-лист для полноты информации в тексте или брифе                            |
| **Star-Story-Solution** | Герой → История → Решение                                     | Сторителлинг: персонаж сталкивается с проблемой → находит выход через продукт |

## 🎯 Стратегический маркетинг

### Анализ и позиционирование

| Фреймворк  | Компоненты (EN)                                                       | Компоненты (RU)                                                                        | Назначение                                                         |
| ---------- | --------------------------------------------------------------------- | -------------------------------------------------------------------------------------- | ------------------------------------------------------------------ |
| **STP**    | Segmentation → Targeting → Positioning                                | Сегментация → Таргетирование → Позиционирование                                        | Разделить аудиторию → выбрать целевую → занять место в её сознании |
| **SWOT**   | Strengths / Weaknesses / Opportunities / Threats                      | Сильные / Слабые стороны / Возможности / Угрозы                                        | Внутренний и внешний анализ бизнеса для стратегии                  |
| **PESTEL** | Political / Economic / Social / Technological / Environmental / Legal | Политические / Экономические / Социальные / Технологические / Экологические / Правовые | Макроанализ среды для долгосрочного планирования                   |
| **5C**     | Company / Customers / Competitors / Collaborators / Context           | Компания / Клиенты / Конкуренты / Партнёры / Контекст                                  | Комплексный анализ рынка перед запуском                            |

### Маркетинг-микс и матрицы

| Инструмент          | Компоненты                                                            | Комментарий                                                                |
| ------------------- | --------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| **7P**              | Product, Price, Place, Promotion, People, Process, Physical evidence  | Расширенная модель 4P: добавляем людей, процессы и «осязаемость» услуги    |
| **BCG-матрица**     | Звёзды / Дойные коровы / Собаки / Вопросительные знаки                | Портфельный анализ: куда вкладывать, что поддерживать, от чего избавляться |
| **Матрица Ансоффа** | Проникновение / Развитие рынка / Разработка продукта / Диверсификация | 4 стратегии роста: от безопасных до рискованных                            |
| **GE/McKinsey**     | Привлекательность рынка × Конкурентоспособность                       | Приоритизация направлений инвестиций по двум осям                          |

### Планирование: SOSTAC

- **S — Situation** (Анализ ситуации)
  _Где мы сейчас? → анализ рынка, конкурентов, клиентов, ресурсов_
- **O — Objectives** (Цели)
  _Чего хотим? → постановка SMART-целей_
- **S — Strategy** (Стратегия)
  _Как достигнем? → сегменты, позиционирование, УТП_
- **T — Tactics** (Тактика)
  _Какими инструментами? → каналы, контент, реклама, бюджет_
- **A — Action** (Действие)
  _Кто, что, когда? → дорожная карта, ответственность, дедлайны_
- **C — Control** (Контроль)
  _Как измерим успех? → KPI, метрики, отчётность, корректировка_

## 👥 Анализ клиентов и пользовательского опыта

| Инструмент        | Расшифровка                                | Суть                                              | Применение                                                                  |
| ----------------- | ------------------------------------------ | ------------------------------------------------- | --------------------------------------------------------------------------- |
| **CJM**           | Customer Journey Map                       | Карта пути клиента                                | Визуализация всех касаний: от первого контакта до лояльности и рекомендаций |
| **JTBD**          | Jobs To Be Done                            | «Работа», которую клиент нанимает продукт сделать | Фокус на задаче клиента, а не на характеристиках продукта                   |
| **RFM**           | Recency → Frequency → Monetary             | Время последней покупки → Частота → Сумма         | Сегментация базы: кто ценен, кто «остывает», кого вернуть                   |
| **Buyer Persona** | Персона покупателя                         | Детальный портрет идеального клиента              | Гипотезы о болях, мотивах, каналах коммуникации для точного таргета         |
| **CAC / LTV**     | Customer Acquisition Cost / Lifetime Value | Стоимость привлечения / Пожизненная ценность      | Баланс: сколько тратим на клиента → сколько он приносит за всё время        |

## 💡 Инновации и дизайн-мышление

### Design Thinking (Дизайн-мышление)

1. Эмпатия → 2. Фокусировка (проблема) → 3. Генерация идей → 4. Прототип → 5. Тест

> Итеративный подход: глубоко понять пользователя → быстро проверить гипотезу

### ТРИЗ (Теория решения изобретательских задач)

| Элемент                                | Описание                                                                         |
| -------------------------------------- | -------------------------------------------------------------------------------- |
| **Анализ противоречий**                | Выявление и устранение конфликтов в системе (например, «дешёво» и «качественно») |
| **ИКР (Идеальный конечный результат)** | Система сама выполняет нужную функцию без затрат и вреда                         |
| **Приёмы разрешения**                  | 40 стандартных приёмов для устранения технических противоречий                   |

### Воронка инноваций

💡 Генерация идей
🔍 Отбор и оценка перспективных
🧪 Прототипирование (быстро и дёшево)
🧪 Тестирование на реальной аудитории
🚀 Запуск на рынок и масштабирование
