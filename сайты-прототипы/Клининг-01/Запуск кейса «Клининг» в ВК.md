# 🚀 Запуск кейса «Клининг» в ВК: План на 2 недели

## Неделя 1: Прогрев и Анонс

### Понедельник — Пост-тизер

**Тема:** «Почему 90% сайтов клининговых компаний теряют клиентов»
**Контент:**

- 3 ошибки с скриншотами конкурентов (размыто)
- Твоё решение (без показа финала)
  **CTA:** «В пятницу покажу прототип без этих ошибок. Подпишитесь, чтобы не пропустить»

### Среда — Видео-презентация (Клип)

**Формат:** Вертикальное видео (30–60 сек)
**Сценарий:**

1.  Экран «До» (серый шаблон) — 3 сек
2.  Переход (свайп) → «После» (твой дизайн) — 5 сек
3.  Демонстрация переключения тем (светлая/тёмная) — 10 сек
4.  Адаптив (телефон → планшет → десктоп) — 15 сек
5.  Финал: «Хочу такой же» + кнопка
    **Музыка:** Трендовая, спокойная
    **CTA:** «Полная презентация — в закреплённом посте»

### Пятница — Основной пост (Кейс)

**Тема:** «Прототип сайта для клининга: 7 слайдов за 3 дня»
**Контент:**

- Карусель из 5–7 изображений (слайды из твоей презентации)
- Краткое описание каждого преимущества
- Ссылка на живое демо (Astro)
  **CTA:** «Бесплатный аудит вашего сайта — в ЛС»

## Неделя 2: Распространение и Холодный Поиск

### Понедельник — Репост в тематические сообщества

**Где искать:** «Бизнес [Твой Город]», «Клининг [Город]», «Франшизы», «Стартапы»
**Что писать:** Не спам! Пиши как эксперт:

> «Сделал разбор типичных ошибок сайтов клининга + показал на примере, как надо. Кому актуально — забирайте бесплатно [ссылка]»

### Среда — Холодные сообщения (аккуратно!)

**Аудитория:** Владельцы клининговых компаний (поиск по группам, бизнес-справочникам)
**Лимит:** 10–15 сообщений в день
**Шаблон:**

> «Здравствуйте, [Имя]! Увидел ваш сайт [название], понравился ассортимент. Я веб-разработчик, недавно делал прототип для ниши клининга — заметил пару моментов, которые могут увеличить конверсию. Могу показать бесплатно, если интересно. Не рекламирую, просто делюсь опытом.»

### Пятница — Пост-отзыв (даже если нет клиента)

**Тема:** «Что я узнал, делая сайт для клининга»
**Контент:**

- Инсайты по нише (какие блоки важны, какие — нет)
- Скорость загрузки (Lighthouse скриншот)
- Адаптивность
  **CTA:** «Ищу 2 проекта на январь со скидкой 20% за кейс»
