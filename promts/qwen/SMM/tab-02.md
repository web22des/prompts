# 🧩 Формулы промтов

| Формула  | Расшифровка (EN)                    | Расшифровка (RU)                         | Когда использовать                               |
| -------- | ----------------------------------- | ---------------------------------------- | ------------------------------------------------ |
| **RTF**  | Role → Task → Format                | Роль → Задача → Формат                   | Когда нужно чётко задать роль ИИ и формат ответа |
| **TAG**  | Task → Action → Goal                | Задача → Действие → Цель                 | Когда важна конкретика: что сделать и зачем      |
| **BAB**  | Before → After → Bridge             | Было → Стало → Мостик                    | Для описания проблемы и запроса решения          |
| **CARE** | Context → Action → Result → Example | Контекст → Действие → Результат → Пример | Для сложных задач, где нужны примеры и условия   |
| **RISE** | Role → Input → Steps → Expectation  | Роль → Вводные → Шаги → Ожидания         | Для пошаговых инструкций с чётким ТЗ             |

# 📐 Структура промта

| Шаг   | Элемент                                     | Что писать                                           | Пример                                                                                                    |
| ----- | ------------------------------------------- | ---------------------------------------------------- | --------------------------------------------------------------------------------------------------------- |
| **1** | **Конкретный запрос**                       | Чётко сформулируй, что нужно сделать прямо сейчас    | «Напиши холодное сообщение для рассылки компаниям»                                                        |
| **2** | **Контекст / Ограничения / Роль / Примеры** | Задай рамки: кто ты, какие условия, что можно/нельзя | «Ты — начинающий веб-разработчик. Стиль — деловой, но простой. Без внешних ссылок. Цель — получить ответ» |
| **3** | **Уточнить интерпретацию**                  | Объясни, как правильно понимать задачу               | «Под "холодным сообщением" имею в виду короткий текст для Авито, не спам»                                 |
| **4** | **Уточняющие вопросы**                      | Попроси ИИ задать вопросы перед выполнением          | «Если чего-то не хватает — спроси: город, ниша, оффер»                                                    |
| **5** | **Идеальный конечный результат (ИКР)**      | Опиши, как должен выглядеть готовый ответ            | «Готовый текст сообщения в 4–5 абзацев, с плейсхолдерами [Имя] и [Ниша], без лишних комментариев»         |
