# ЗАДАЧА:

Что есть:

Файл `index.html` или (`.mhtml`) — сохраненная страница ВКонтакте с моими постами

**Задача** - извлечь из этого файла данные о каждом посте

Что нужно получить:
Для каждого поста:

- Дата публикации
- Количество просмотров
- Количество лайков
- Количество репостов
- Количество комментариев

Как это должно работать:
Запускается скрипт вручную

1. Скрипт читает файл
2. Скрипт создает новый файл (формат нейминга `name-files_date.js`) с результатами (`JSON`), только нужные строки
3. Скрипт должен делать подробное логирование
